Data Description &
General Structure

The dataset used in this project comes from National Heritage Academies (NHA) and captures the applicant journey across the enrollment flow. The data spans multiple academic years (2023–2025) and includes over 200,000 applicant records and 60 columns.

Data Access Statement Data supporting this analysis cannot be made available due to client confidentiality and NDA restrictions. Code and methodological details are provided to enable replication with comparable data.

Key data dimensions
Record Structure
Each row represents one application record
Applicant Information
Student identifiers, grade level, and application year.
Application Timeline
Key timestamps such as application submission, offer status, and enrollment decision.
Enrollment Outcomes
Detailed status indicators capturing each stage of a student’s behavior through the enrollment pipeline, including Applied, Waitlisted, Accepted, Registered, Enrolled, and post-enrollment outcomes such as Re-Enrolled, Withdrawn, and Graduated. These statuses reflect both intermediate decision points and final enrollment outcomes, enabling analysis of applicant transitions and drop-off points throughout the process.
Family Context
Features such as whether the applicant has siblings currently enrolled.
School Attributes
School location and related contextual information.
